SharpEmu now has a proper home on the web. Alongside the project introduction
you'll find the piece we're most excited about: a **compatibility database
tracking every PS5 title** — 8,700+ games imported with their title IDs, cover
art and metadata.

Every game has its own page where testers can report how far it gets on the
standard ladder (*nothing → boots → menus → ingame → playable*), with the exact
emulator build, OS and hardware noted. Comments on each game page are coming
soon.

## Help us fill it in

The database is only as good as its reports. If you've run a game on SharpEmu:

- open a [compatibility report](/compatibility/report) — it takes five minutes,
- or send a pull request adding a markdown file to the site repository.

As always: test with dumps of games you own. SharpEmu remains an early,
experimental research project — most games don't run yet, and that's exactly
why honest reports matter.
